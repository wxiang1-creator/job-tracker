# 求职申请管理看板 Job Application Tracker

一个面向大学生求职场景的求职申请管理系统，用于统一管理岗位投递信息、截止日期、面试安排和申请状态，帮助用户更清晰地掌握整体申请进度。

## 项目背景

大学生在求职申请过程中，通常需要同时关注多个岗位的截止日期、材料提交状态、笔试和面试进度。如果仅依赖招聘平台逐个查看，会带来较高的信息管理成本。

因此，本项目设计并实现了一个“求职申请管理看板”，帮助用户集中管理所有岗位申请记录，提高求职阶段的信息整理效率。

## 项目目标

本项目主要解决以下问题：

- 统一记录不同公司的岗位申请信息
- 可视化展示当前申请状态
- 管理截止日期与面试时间
- 帮助用户快速了解整体求职进度

## 核心功能

### 1. 新增申请记录
用户可以添加新的岗位申请信息，包括：

- 公司名
- 岗位名
- 投递日期
- 截止日期
- 当前状态
- 面试时间
- 备注

### 2. 查看申请列表
所有岗位申请会以卡片形式展示，信息更直观。

### 3. 状态管理
支持以下状态：

- 已收藏
- 已投递
- 笔试
- 面试中
- Offer
- 已淘汰

用户可以随时更新申请状态。

### 4. 筛选功能
支持按状态筛选申请记录，方便快速查看特定阶段的岗位。

### 5. 数据统计
顶部统计区可以展示：

- 总申请数
- 已投递数
- 面试中数
- Offer数

### 6. 进度可视化
通过进度条展示当前整体求职推进情况。

### 7. 数据持久化
后端会将数据保存在本地 `data.json` 文件中，刷新页面后数据不会丢失。
